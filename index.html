We will use ffmpeg to compress images and videos. The compression happens in real time, and you can download the file as soon as the upload is complete. Supported format include: pptx, docx, odp, odt

In general, compression does not cause any content and format changes but greatly reduce file size. We will do the following things:

  • Scan all media in PPTX
  • Use ffmpeg to transcode images and videos
  • Perform audio normalization for video

This program is just a toy. No information is collected. Files are not saved on the server. For more details, you can check the source code.
